## السياق

مرةً كل شهر، أكسر كل شيء (جهازي الشخصي وبيئة المختبر الخاصة بي) لأعيد بناءه « حجراً بحجر ».

سواء كان تثبيت التطبيقات على جهازي عبر PowerShell، أو إعادة إنشاء وكلاء الذكاء الاصطناعي في Copilot أو ملفات تعريف Intune Full Cloud Autopilot.

لمتابعة هذه الدورات، أنشأت هذا الخط الزمني لتوثيق كل إجراء تم اتخاذه خلال كل:
« رجوع / إعادة ».

## قوس الطالب
import { Callout } from 'fumadocs-ui/components/callout';

<Callout type="warn">
**مهم**: كل إجراء أو مهارة جديدة يتم تعلمها في دورة ما سيتم تكرارها وتحسينها في جميع الدورات اللاحقة. لا يوجد شيء يُعتبر منتهياً حقاً — كل شيء يُعاد تشغيله وتحسينه وإعادة بنائه من الصفر في التكرار التالي.
</Callout>
<Steps>

<Step>
### حساب Azure ترخيص مجاني (الدورة الأولى) ###
لفهم Azure Active Directory بشكل أفضل، قررت إنشاء tenant خاص بي وإعداد مختبر Azure AD.

هذا سمح لي باستكشاف خصائص Azure AD مثل إدارة المستخدمين، والمزامنة مع Active Directory محلي، والتكامل مع تطبيقات سحابية.
</Step>

<Step>
### تهيئة الحاسوب وPowerShell (الدورة الثانية) ###
حصلت على قرص SSD جديد بسعة 2 تيرابايت لحاسوبي المحمول، لذلك تحديت نفسي بكسر كل شيء وإعادة بناءه بالكامل على القرص الجديد عبر PowerShell.
</Step>

<Step>
### تعلّم Intune وإنشاء الدليل (الدورة الثالثة) ###
قمت بتجربة 30 يوماً من M365 Business Premium لتعلم Intune وفي نفس الوقت: إنشاء « دليل مسؤول / Handbook » لتوثيق عملية تعلمي.

Commits بتاريخ 19 أبريل 2025 => [github/apescasio](https://github.com/apescasio/memo.apescasio.fr)
</Step>

<Step>
### إنشاء وكلاء Copilot Agents AI Studio في « LabOfApes » (الدورة الرابعة) ###
انزعجت لأن مايكروسوفت دفعت تغييرات إلى الواجهة الرسومية لـ Copilot Studio، مثال: واجهة المُشغّل تغيرت بالكامل، فصرت تائهاً.

انتهى بي الأمر بالالتزام وشراء M365 Business Premium + إضافة Copilot Studio AI Agent (بمجموع 75$ شهرياً). في tenant « LabOfApes » أنشأت ISO (ذكاء اصطناعي لقسم تكنولوجيا المعلومات)، وأخطط لإنشاء MIDL (ذكاء اصطناعي لقسم الموارد البشرية).
</Step>
<Step>
### ART, ALZHEIMERS-RESISTANCE-TRAINING، تطبيق Python (الدورة الخامسة) ###

[Using Python to Train Alzheimer’s Resistance (ART), Memory, WPM - Insignificant Student (1/3000).](https://www.youtube.com/watch?v=_bB--9vh2CY)

لست طبيباً، هذه ملاحظاتي الخاصة، أنا آرون (Iso) بيسكاسيو، طالب بسيط.

في أيام الثانوية، كنت أعيد كتابة ملاحظاتي لمادة التاريخ والجغرافيا وغيرها على لوحة المفاتيح مراراً وتكراراً.

بعد سنوات، أنشأت هذا التطبيق بلغة بايثون لتدريب سرعة الكتابة (كلمات بالدقيقة) وذاكرتي.

أكتب بأقصى سرعة ممكنة، وأثناء الكتابة أكرر كل جملة في ذهني.
</Step>
</Steps>
